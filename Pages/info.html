Trex-Fold

Instructions

Please follow these steps to run the game -

  1. Open a new tab in Chrome.

  2. Enter " chrome://flags/ " into the address bar.

  3. Locate Experimental QUIC protocol.

  4. Using the drop-down menu, select Disable.

  5. Restart Chrome.

  6. Allow camera permission if prompted.

  7. Restart browser if necessary.

How to play the game?

  1. Hold your stretched palm in the view-finder.

  2. Wait for a few minutes until you see red markers appear above the video.

  3. Make sure you are sitting in a well lit room.

  4. Fold your palms to play the game !
