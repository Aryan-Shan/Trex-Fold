# Trex-Fold
Chrome T-Rex game with TensorFlow.Js
1. Project Overview
Trex-Fold is a browser-based, gesture-controlled reimagining of the iconic Google Chrome offline T-Rex dinosaur
game. Instead of pressing the spacebar, players use their webcam to control the dinosaur by folding their palm
toward the camera. The project combines a classic arcade-style game with real-time computer vision — all running
entirely in the browser with no server or installation required.
The live demo is available at aryan-shan.github.io/Trex-Fold and the full source code is hosted on GitHub.
2. Objectives
• Reinvent the classic Chrome T-Rex game with a contactless, gesture-based control scheme.
• Leverage TensorFlow.js and the HandPose model to perform real-time hand landmark detection in the browser.
• Demonstrate a practical application of machine learning on the client side — no backend or Python runtime
needed.
• Provide an accessible, plug-and-play experience: open a URL and play immediately using any webcam.
